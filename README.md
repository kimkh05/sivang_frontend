## Project Setting

First, Let's clone this repository.

```shell
git clone https://github.com/kimkh05/sivang_frontend.git [dir] # Frontend Repository
```

## Install node_modules
```shell
yarn
```

## Getting Start

```shell
yarn dev
```

React, Vite, Typescript, Yarn, Styled-Components, Axios, Recoil
